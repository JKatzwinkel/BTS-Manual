Initial configuration of BTS

BTS installer currently does create BTS entries in the windows "Start" menu or on the desktop. You can manually create a shortcut of the "bts.exe" file and drag-and-drop it into the start menu and onto the desktop. Doubleclick this icon to start the program.

Configure your BTS

After the initial BTS start, a few settings have to be made. In the main programm window click on "Preferences / Preferences/ Berlin Text System General":

  1. In the "Corpus Settings" add the available corpora to the list of the "Active Corpora", selecting all corpora you are going to work with.
  2. The checkbox "Activate to select main working corpus" is deactivated per default. After selecting this option all new objects will be saved in the "Main working corpus", regardless of your current position. Activate this feature only if you need it.
  3. Click "Corpus Navigator Settings": activate "Corpus Navigator sort by key". Press "Apply" to save the setting. Repeat the same for "Lemma Navigator Settings" and "Thesaurus Navigator Settings".
  4. "Text Editor" -> "Sign Text Editor": activating "Show Hieroglyphs", "Show Lemma ID", "Show Flexion" and "Show Translation" (language you will work with) are recommended to use the Sign Text Editor properly.
  5. Press "Apply" to save all changes and close the window by clicking "OK".

    To be able to use BTS one final step has be done - the configuration of the keyboard, which will be used for text transliteration.
