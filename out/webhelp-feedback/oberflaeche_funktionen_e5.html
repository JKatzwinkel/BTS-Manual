Common features

BTS-workspace consists of dynamic windows, which can be hidden, moved and resized. Each window can contain one or more tabs. Tabs can be moved from one window to another or opened in a separate window.

The icons common for all viewing modi are the following:

: minimizes window (places the window icon on the icon bar on the left)

: maximizes window. All other windows are minimized

: restores window to the regular (non-maximized) view

Each tab in the each view mode can be grouped together with other tabs or in the separate window. For this, left-click on a tab and drag & drop it to the necessary position. "Transliteration", "Sign-Text-Editor" and "JSesh View" are parts of "Text editor" tab and connot be moved.

The arrangement will be saved for the current viewing mode ("Corpus", "Thesaurus" and "Lemma"). Restarting the programm will undo the changes and restore the default windows arrangement.
