Installing and configuring the keyboard

To be able to enter the transliteration in BTS you need to install a special keaboard layout.

  • Install Tavultesoft Keyman Desktop

Keyman Desktop is a free software, which manages the keyboard. To download it go to http://keyman.com/desktop/download.php, scroll down to "Download Keyman Desktop 9.0 without any keyboards" and click on "Download now". Follow inscructions on the screen for installing the programm correctly.

Install "BBAW Libertine" Font

BTS works with "BBAW Libertine" font, which has to be present in your system. For this in the programm package right mouse-click on "BBAWLibertine_ah.ttf" and select "install". Check is the font has been installed in C:\Windows\Fonts.

  • Install the AAeW Keyboard

Now the "AAeW Keyboar"d can to be installed in the Keyman Desctop. This keyboard layout has all necessary characters, including brackets, verse points and cartouches.

In the bottom task bar click on "Tavultesoft Keyman Desktop" icon and select "Configuration". Click on "Install keyboard..." and select fileЕ"aaew.kmx" available in your program package. Click "OK" to finish the installation.

NOTE: The "Onscreen Keyboard" function is not available yet. See "Keyboard layout" section for the position of the characters.

  • AAeW Keyboard characters

The lower left characters are entered by simply pressing the key.

The upper characters are entered by pressing Shift + Key

The lower right characters are entered by pressing Ctrl + Alt + Key. Example: "Ctrl" + "Alt" + "8" = "["

Characters such as "ẖ" are generated by "deadkeys". You type these signs in by combining the elements "-" and "h" respectively. For example to write "ẖ" press the deadkey (always first) for "-" ( "Shift" + "-") and then "h".
