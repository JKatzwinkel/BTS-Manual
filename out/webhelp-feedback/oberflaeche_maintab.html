Main Tab

The text-input fields and drop-down menus in this tab give you diverse options to enter metadata for your text or object. If you are working on a "child" object and want it to have the same metadata as the "parent" object, you can click "Inherit Passport Data" in the top right corner of the Passport Editor.

  • Name: change the name of your object or text.
  • Type and "Subtype": choose a type and subtype (e.g. text, object, etc.) as defined in Edit Configuration / objectTypes.
  • SortKey: define the sortkey to rearrange the texts/objects in the Corpus Navigator tree (the default order is alphabetical).
  • ID: displays the ID of the object, which was generated automatically. It cannot be changed
  • Author: displays the autor (creator) of the object.
  • Date: displays the date of creation of the object.
  • Visibility: define who can see the object or Text (public, project, group, user).
  • Review State: enter or edit the Review state of the object or text (published, reviewed, awaiting-update, awaiting-review, new).
  • Project and Corpus: clicking on you can assign the object or text to another project / corpus. Note that only the physical position of the object will change (displayed in Navigator in the square brackets), not its position in the hierarchical tree of objects.
