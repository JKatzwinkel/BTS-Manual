Passport Editor

In the default view, the "Passport Editor" is located in the lower centre of the BTS user interface. Here you can create a "passport" for the lemmata/objects by entering their metadata. The "Passport Editor" is divided into different categories which are organized in tabs inside the "Passport Editor" window. For "Lemma" these tabs are: Main, Relations, IDs, Definition, Bibliography, Lemma plus the toolbar.

Toolbar

Clicking opens the Menu (see Common features) containing the following entries:

Inherit Passport Data

This function can also be triggered by leftclicking . It transfers the data from a higher level lemma to a lower level (root entry to child entry).

Inherit Overwrite Passport Data

Using this option overwrites passport data if data has already been commited before.

Main tab

Main tab contains general information about the lemma entry.

Name: Choose or change the name of the lemma.

Type and Subtype: Choose a type and subtype (e.g. substantive, adjective etc.).

SortKey: Rearrange the texts in the Corpus Navigator tree (the default order is alphabetical).

ID: See the ID of the lemma.

Author: See and edit the author of a lemma.

Date: See the date of creation of the lemma.

Visibility: Change who can see the lemma (public, project, group, user).

Review State: Enter or edit the Review state of the lemma (published, reviewed, awaiting-update, awaiting-review, new etc.).

Project: move the lemma to another project.

Corpus: See or edit the name of the corpus. Click to move an object/lemma between projects/databases.

Relations tab

This tab provides information on the relation between the objects.

Relation: Choose differnt types of relations, including "Partof", "referencing" and "referencedby".

Object: Enter the Name of the related object here.

Search Object: Search for the object in the corpora of your current project, so that you do not have to enter it manually.

Add Relation: Click it to add more relations.

Remove Relation: This removes a relation from the Object.

IDs tab

Since there is the possibility that the text appears in extern projects the link between those extern projects and the one you are working on can be seen here.

Provider: Choose the provider from the drop-down menu.

Type: Enter the type of the ID.

External ID: If IDs exist due external projects, e.g. Trismegistos or other, then the ID of those external projects can be entered here.

Add Identifier: Click this to add another set of information here.

Definition tab

This tab is to define your object and record your working process. Editor's comments can also be added here. Use the provided input fields:

  • Definition: Define the type of your lemma here.
  • Comment: Comment your work.

Bibliography tab

Enter a bibliographical information using the Thesaurus here. The tab contains:

  • "Bibliographical text field"
  • "Bibliographical entry/ Bib. item": navigate through Thesaurus with Search to find the correct bibliographical reference.
  • "Bibliographical entry/plates": enter the plate number and select the check box with a letter below.
  • Comment: use this input field to comment your reference.
