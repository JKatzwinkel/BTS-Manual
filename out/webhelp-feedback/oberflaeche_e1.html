UI Reference

This section provides the description of the BTS User Interface. The BTS program window is split into the following areas.

At the top there is the Menu bar (highlighted on the picture below with dark blue) and the Toolbar (pink).

Below there is theWorkspace, which is divided into four parts representing the program's elements: Navigator on the left (which, depending on the view selected, displays Corpus-, Thesaurus- or Lemma-Navigator) (maked green on the screenshot) and Annotation on the right (dark red). In the top centre there is the "Text Editor", containing tabs "Transliteration", "Sign Text Editor" and "JSesh View". At the bottom center there are three tabs: "Passport Data Editor", "Lemmatizer" and "HTW". Note that as soon as the text is selected in the Navigator window, the titles of the tabs "Text Editor", "Passport Data Editor" and "Lematizer" change to the text name.

At the bottom of the programm window the "Status bar" is displayed

Content-related organization: Workspace view

BTS Workspace can have three different viewing modes: Corpus, Lemma and Thesaurus. Switching the view from one to another loads the associated content-elements (Navigator, Editor, Data, Commentary) into the workspace.

Optical Organisation: Windows and Tabs

The elements inside the workspace are organized in windows and tabs. These two levels of the UI are dynamic, which means you can regroup the tabs inside the window, dragging and dropping them to the desired position. Note that after programm restart the default view is restored.
