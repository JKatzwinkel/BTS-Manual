Update / Uninstall BTS

  1. Update BTS

    BTS automatically looks up for updates everytime you start the application. In case an update is found, you will be offered a choice to install it or to postpone it. The update can be intalled from the programm UI as well. For that in the top menu bar click on "Help" / "Update Application".

  2. Uninstall BTS

    BTS is not listed in the "Control Panel \ Programs \ Programs and Features" section thus it is not possible to uninstall it in the regular way. To uninstall the application, close the programm window and delete all files from the installation folder.

    NOTE: You might need to restart the PC before deleting the folder. The reason for this is that the DB services might be running and might block the deletion.

    NOTE: Using "Help" / "Uninstall software" entry in the upper menu bar of BTS will not be able to uninstall the application. It only uninstalls separarte components. completely, because Database service was started and is running. You will have to close the programm window and delete the files again.
