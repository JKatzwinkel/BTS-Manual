Introduction

Welcome to Berlin Text System V 3.0.ш- a new version of the software specially designed for annotating Ancient Egyptian texts! Its three major components - Text Editor, Lemma List and Thesaurus - work together to guarantee the extensive word processing of Ancient Egyptian.

Along with the tools for the transcription, translation and hieroglyphic encoding, BTS enables the annotation of word forms, like adding a commentary or marking a rubrum or citation.

Metadata (ex. bibliography, date) of the texts can be added, using the built-in Thesaurus.

The Lemmatizer assigns each word in the text to the existing lemma, it is also possible to create new lemma entries.

The backend and the user interface of the software has been completely redesigned for the new version, including the new database structure (Couch DB), data format (JSON) and search engine (Elasticsearch).

This User Manual provides the overview of the features now available in BTS V.3.0. It is split into two sections: "UI Reference" and "How to".

UI Reference section provides a detailed description of the User Interface.
