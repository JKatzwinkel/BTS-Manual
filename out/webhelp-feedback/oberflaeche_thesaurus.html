Thesaurus

The "Thesaurus" is an optional part of the workspace. It can be activated by clicking the "Thesaurus" icon in the Toolbar. Thesaurus contains a uniform terminology, which can be used for object's metadata (Passport data). New entries can be added as well.

The three windows from left to right are: Thesaurus Navigator, Passport Data Editor, Annotations

  • Thesaurus Navigator

  • Passport Editor

  • Annotations
