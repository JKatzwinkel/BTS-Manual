3. Lemmatizing and encoding inflections

To lemmatize a transliterated text, the lemmatizer is used. It can be found in the central lower part of the default Text Corpora view.

Only white marked objects in the Sign Text Editor are lemmata and can be lemmatized.

Click the second tab "Lemmatizer" in this part of the screen and select the "Sign-Text Editor" tab above in the Text Editor window.

You can see several elements:

  1. "Activate" button: activates / deactivates other fields
  2. The first display field "Selected Word" displays the transliteration of a lemma that you have activated by clicking it (in our example: pn). In this example it has already been lemmatized, so the other fields are filled. If you find out that you have made an error in the transliteration, you can modify it in the "Selected Word" field and it will be altered in the Sign-Text-Editor and the Transliteration tab automatically.
  3. "Lemma" provides the lemma number from the WL (word list). Next to it, the selected transliteration of a lemma is displayed. To lemmatize a lemma, click on the correct suggestion in the field below. The default setting in "Preferences / Preferences / Lemmatizer" allows you to activate the function "automatically select the first lemma proposal".
  4. "Flexion" shows the code referring to the grammatical form of the inflection of the selected word.
  5. In the field "Translation" you can choose one or several fitting translations from the lower field (in the example there is only one option, but there can be more). Make sure to have selected the right language in the drop-down menu to the right. Your default language is the one connected with your login data. Choose several available translations by left clicking and holding CTRL at the same time. If none of the options seems fitting to you, you can enter your own translation as well.

If you cannot find a lemma, click the magnifying glass and search for it in the lemma list by entering the WCN or the name of the lemma. You can also reduce the search results by using the filters "Search for IDs" or "Search for Names only". Additionally you can automatically add wild cards / quotation marks by clicking the provided buttons in the Search pop-up. More on search function see the corresponding section. If the lemma you need does not exist please contact the person who is responsible for the lemmata-list in Berlin.

To go to the next lemmatizable lemma, either click "Confirm current lemma editing and continue to next lemma" or ">" on the right above the lemmatizer. ">|" brings you to the end of the line and >|| to the end of the text.

Deleting a lemmatization can be engaged by clicking "Remove lemma information" in the Toolbar of the "Lemmatizer".
