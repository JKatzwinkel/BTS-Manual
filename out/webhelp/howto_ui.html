Introduction to BTS User Interface

This chapter provides the brief introduction to the BTS User Interface (= BTS UI).

The BTS program window is split into four major sections:

  • Menu bar
  • Toolbar
  • Workspace
  • Status bar

The Menu bar and the Toolbar are located at the top of the program window. The area below them is called Workspace. The contents of the Workspace area change depending on the viewing mode. BTS has four such modes, activated by clicking a relevant button on the Toolbar:

Text Corpus view

Lemma view

Thesaurus view

Abstract Text view

Within each view there are sections with content. Different sections can be grouped into sub-sections (e.g. if they refer to one heading section). If there is more than one section or sub-section in the same position, then they are grouped in tabs. Additionally, those individual sections or sub-sections can be dragged and dropped into other positions of the workspace or outside of the BTS program window.

The default arrangement of the sections is currently the following (see also screenshot below):

  • Navigator tab on the left

    Depending on the view selected it can be called Corpus-, Thesaurus- or Lemma Navigator. The sub-tabs are the following: "Corpus Navigator": "Text Corpora" + "Bin"; "Lemma Navigator": "WL" + "Bin"; "Thesaurus Navigator": "Thesauri" + "Bin"; Abstract Text Navigator": "Ths" + "Bin".

  • Annotation and Translation tabs on the right

    In "Lemma" and "Thesaurus" view the translation tab is not displayed.

  • Editor tab at the center top

    In Text Corpus view this tab is called "Text editor" and contains three sub-tabs: "Transliteration", "Sign Text Editor" and "JSesh View". In Lemma view it is called "Lemma editor" and has no sub-tabs. Abstract Text view contains both "Text editor" with its three sub-tabs and "Abstract text editor" tab.

  • At the bottom the three tabs are placed: Passport Data Editor, Lemmatizer and Hieroglyph Type Writer (HTW)

    The Text Corpus and Lemma views display all three. Thesaurus and Abstract text views have only Passport Data Editor. Note that as soon as a text or an object is selected in the Navigator window, the titles of the tabs Text Editor, Passport Data Editor, Lemmatizer and Translation change to the text name.

At the bottom of the program window the Status bar is displayed.

UI glossary

This manual uses the following terms adopted from the "Glossary of UI terminology" by Microsoft.

  • Menu bar / drop-down menus / drop-down entry

  • Toolbar / toolbar button

  • Icon / ToolTip

  • Workspace / View (workspace mode)

  • Navigation tree / Node (an entry in a navigation tree, which can be a text corpus, text corpus object or a text)
