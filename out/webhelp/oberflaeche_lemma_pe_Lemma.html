Lemma

Lemma tab provides information about different properties of the lemma. These are the following:

  • Comment: Any comment to the Lemma which did not fit in any other subsection.
  • Comment (English translation): Any comment to the English translation of the Lemma.
  • Lsort, arb_vermerk, simplify, wclassnum, woart, word_class: present information tags inherited from the previous BTS version.
