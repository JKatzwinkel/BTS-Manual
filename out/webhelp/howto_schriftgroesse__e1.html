Changing font size

Changing font size of the BTS is quite simple. Select Window entry from the Menu bar. The first option in the drop-down menu will be "Switch Font Size/CSS Theme". Click on it in order to enlarge the font. There is no list of font sizes to choose from. The font size will be enlarged or shrank upon clicking on the entry. Clinking on the entry the second time will bring back the original size and change the look of the BTS a little. Next two clicks will keep enlarging the font with the new coloring. The next clicks shrink it again and bring back the original coloring. Test it out in order to establish which look is the best for you.
