Findspot

Another tab specific to an object, serves to enter information about the findspot of the object.

This tab is divided into two sections: "Place" and "Former Place".

Place

  • Place: Choose the findspot from the Thesaurus.
  • Comment: Add a comment to the findspot.
  • is_origin: Select this check box if the chosen findspot is the original site where the object was created.
  • Certainty: Choose between "Certain", "Probable" and "Uncertain"

Former place

  • former_place: Select a previous find spot from the thesaurus.
  • Comment: Add any comment to the findspot.
  • is_origin: Add the origin of the object if the find spot is not equal to the original location.
  • Certainty: Choose between "Certain", "Probable" and "Uncertain".
