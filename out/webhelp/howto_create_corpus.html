Creating / modifying a corpus

The following function is available to the user with global administrator privileges only. The detailed description of the UI elements can be found in Text Corpus section.

Creating a corpus

  1. Click the New Corpus icon. It is located above the Corpus Navigator. Alternatively right mouse-click on any corpus and select Create New Text Corpus.
  2. Choose a name and a prefix (a prefix is a short name under which the corpus will be saved in the database). No special signs, spaces or capital letters are allowed in the prefix!
  3. Select the check box if you want to "Synchronize corpus with central database". Other users will be able to see your new corpus only if it is synchronized. Otherwise, it will rest in offline mode on your PC. After you have finished, click "OK". Cancel the creation with the "Cancel" button.
  4. Your new corpus will appear in the corpora list, in alphabetical order.
