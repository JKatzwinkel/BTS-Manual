IDs

Since there is the possibility that the text appears in external projects, the ID tab provides the opportunity to enter the external ID, thus linking the projects. By default a button will be displayed. Click it to add an ID.

  • Provider: Choose the provider (an external project) from the drop-down list.
  • Type: Enter the type of the ID, in case such exists.
  • External ID: Enter the external ID here (e.g. Trismegistos).
  • Add Identifier: Add another set of information, including the "Provider", "Type" and "External ID"
