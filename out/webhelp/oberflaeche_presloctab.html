Present Location

This is another tab specific for an object.

  • Location: Choose the location of your object from the Thesaurus.
  • Inventory Number: Enter the number of your object if provided.
  • Comment: Enter a comment about the location of the object if necessary.
  • in_situ: Select this check box if the object is still preserved in its original location.
  • is_present_location: Select this check box if the present location differs from its original location (e.g. on display or in inventory of a museum).
  • Add widget: Click to add an additional "location" section so you can enter a previous location (if it differs from the present location).
  • Delete widget: Click to delete a section.
