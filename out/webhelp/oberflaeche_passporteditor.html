Passport Editor

The Passport Editor section is located in the bottom part in the middle of the Text Corpus view. It contains the metadata of a selected text corpus object. The "Passport Editor" is divided into different subsections which are organized in tabs. These categories vary depending on the view: Text Corpus, Lemma, Thesaurus or Abstract Text. The rights to modify object's passport data depend on your user role for the selected corpus.

  • Main Tab

  • Relations Tab

  • IDs Tab

  • Definition Tab

  • Findspot Tab

  • Present Location Tab

  • Date Tab

  • Object Tab

  • Text Tab

  • Bibliography Tab

  • Synonyms tab
