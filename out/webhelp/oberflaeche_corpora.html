Text Corpus View

Text Corpus is the default view of the BTS workspace. Here you can work on text corpus objects and texts, including transliteration, translation, lemmatization, hieroglyphs and their metadata.

There are following tabs to work with:

  • Corpus Navigator (default position: left part)
  • Text Editor (default position: middle part, top)
  • Passport Data Editor (default position: middle part, bottom)
  • Lemmatizer (default position: middle part, bottom)
  • Hieroglyph Type Writer (HTW) (default position: middle part, bottom)
  • Annotation and Text translation (default position: right part)

NOTE: in the default view the tabs "Passport Data Editor", "Lemmatizer" and "HTW" are grouped together in one window. Nevertheless they can be moved separately.

As soon as you select a text in the Corpus Navigator, titles of these tabs as well as those of the "Text editor", "Annotation" and "Text Translation" change to the text title.
